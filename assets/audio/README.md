# Audio Assets

This directory contains audio files for the Mario RL training dashboard.

## Required Files

Place the following audio files in this directory:

| File | Description | Suggested Source |
|------|-------------|------------------|
| `music_theme.mp3` | Background music (8-bit style) | OpenGameArt.org or similar |
| `sfx_death.mp3` | Death sound effect | NES Mario death sound |
| `sfx_level_complete.mp3` | Level completion fanfare | NES Mario level clear |
| `sfx_coin.mp3` | Coin collection sound (optional) | NES Mario coin sound |

## Free Audio Sources

- [OpenGameArt.org](https://opengameart.org/) - Free game audio
- [FreeSound.org](https://freesound.org/) - Creative Commons audio
- [8-bit Music Generator](https://www.udio.com/) - Generate 8-bit style music

## Notes

- Files should be in MP3 format for browser compatibility
- Keep files small (< 1MB each) for fast loading
- The dashboard will work without audio files (just no sound)
